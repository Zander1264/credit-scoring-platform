# Scoring Service

Сервис оценки кредитоспособности клиентов. Принимает данные пользователя и предлагает подходящие кредитные продукты с учётом его финансового профиля.

## 📌 Функции

- **`POST /pioneer`** — обработка заявки от нового клиента:
  - Анализ возраста, дохода, занятости, наличия имущества
  - Расчёт скорингового балла
  - Формирование одобренных продуктов

- **`POST /repeater`** — обработка повторного заёмщика:
  - Проверка истории выплат
  - Ускоренное решение на основе прошлого поведения
  - Подбор условий с лояльными ставками

## ⚙️ Технологии

- **FastAPI** — REST API, валидация через Pydantic, Swagger UI
- **Kafka** — публикация событий (например: "scoring_completed")
- **Redis** — кэширование результатов проверок
- **Docker** — контейнеризация
- **Kubernetes + Helm** — развёртывание и масштабирование

## 🧪 Тестирование

- Покрытие юнит- и интеграционными тестами (`pytest`)
- Проверка граничных значений (доход, возраст и т.д.)
